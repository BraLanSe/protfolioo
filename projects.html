Application de Messagerie

Une messagerie en temps réel avec WebSocket, annotation d’émotions, AJAX et PHP.

Fonctionnalités :

  • Système d'inscription sécurisé avec vérification par email (PHPMailer).
  • Connexion et gestion des sessions utilisateur.
  • Envoi et réception de messages en temps réel via WebSocket.
  • Annotation des messages avec des émotions (positif, neutre, négatif).
  • Interface utilisateur responsive avec AJAX pour mises à jour sans rechargement.
  • Stockage des messages et annotations dans une base de données PostgreSQL.

Technologies : PHP, JavaScript, WebSocket, AJAX, PostgreSQL, PHPMailer, HTML5, CSS3.

Projet SAÉ S4 - Annotation de Messages

Application web d’annotation des messages avec des émotions, utilisée pour le service client.

Description : Développement d’un outil permettant aux utilisateurs de classifier les messages selon leur émotion exprimée afin d’améliorer la qualité du support client.

Fonctionnalités :

  • Interface intuitive pour annotation rapide.
  • Visualisation des statistiques par émotion.
  • Export des données annotées au format JSON.

Technologies : PHP, JavaScript, AJAX, PostgreSQL, JSON.
