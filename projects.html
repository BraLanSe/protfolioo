Application de Messagerie

Une messagerie en temps réel avec WebSocket, annotation d’émotions, AJAX et PHP.

Fonctionnalités :

  • Système d'inscription sécurisé avec vérification par email (PHPMailer)
  • Connexion et sessions utilisateur
  • Messagerie instantanée via WebSocket (sans rafraîchissement)
  • Interface dynamique en AJAX et JavaScript
  • Annotation automatique des émotions d’un message à l’aide d’un modèle de classification (émotions positives, neutres, négatives)
  • Design responsive et personnalisé

Gestion de Notes

Projet académique en SQL : normalisation, vues matérialisées, données JSON, types composites, agrégats.

Contenu :

  • Modélisation de la base : relations entre étudiants, modules, notes
  • Utilisation de vues pour simplifier la consultation
  • Calculs statistiques avec fonctions SQL (moyennes, max, min par module)
  • Types JSON pour stocker les détails des évaluations
  • Création de fonctions PL/pgSQL pour automatiser les calculs

Simulation de l'Algorithme du Simplexe

Implémentation manuelle de l’algorithme du simplexe en Python, avec toutes les étapes détaillées.

Objectifs :

  • Comprendre le fonctionnement du simplexe à deux phases
  • Développer un algorithme pas à pas sans bibliothèque externe
  • Affichage des tableaux intermédiaires pour chaque itération
  • Résolution de cas complexes d’optimisation linéaire
  • Utilisation en terminal avec saisie des données via fichiers ou interface
