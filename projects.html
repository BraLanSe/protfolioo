Application de Messagerie & Annotation d’Émotions

Messagerie instantanée en temps réel avec annotation des messages par émotions, utilisée pour améliorer le service client.

Fonctionnalités :

  • Système d'inscription sécurisé avec vérification par email (PHPMailer).
  • Connexion et gestion des sessions utilisateur.
  • Envoi et réception de messages en temps réel via WebSocket.
  • Annotation des messages avec des émotions (positif, neutre, négatif) grâce à une interface intuitive.
  • Visualisation des statistiques d’émotions et export des données annotées au format JSON.
  • Interface utilisateur responsive avec AJAX pour mises à jour sans rechargement.
  • Stockage des messages et annotations dans une base de données PostgreSQL.

Technologies : PHP, JavaScript, WebSocket, AJAX, PostgreSQL, PHPMailer, HTML5, CSS3.

Projet de Première Année – Base de Données avec MariaDB

Création et peuplement d’une base de données relationnelle avec MariaDB.

Objectifs : Concevoir une base de données normalisée, créer les tables, insérer des données et réaliser des requêtes SQL.

Fonctionnalités :

  • Modélisation conceptuelle et logique des données.
  • Création des tables avec contraintes d’intégrité.
  • Peuplement de la base avec des données test.
  • Rédaction de requêtes SQL pour extraction et analyse des données.

Technologies : MariaDB, SQL, modélisation UML.
